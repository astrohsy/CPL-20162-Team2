# ScanYou Team 규칙

* 같은 파일을 같이 작업하지 않는다.

* 새로운 작업하기 전에는 '꼭' github로 부터 pull을 받는다. (Console에서 "git pull" 입력)

* 시작할 때
    > git clone https://github.com/astrohsy/CPL-20162-Team2.git

* 프로젝트를 최신 상태로 갱신할 때
    > git pull

* 내 소스를 github로 업데이트 할 때
    > git commit -am "수정된 내용" <br />
    > git push master origin
